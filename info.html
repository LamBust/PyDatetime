Библиотека datetime — работа с датой и временем

Библиотека datetime предназначена для работы с датой и временем в Python. Она позволяет получать текущую дату и время, сравнивать даты между собой, а также вычислять временные промежутки. Эта библиотека часто используется в программах, где важно фиксировать время событий.

С помощью datetime можно форматировать дату в удобный для пользователя вид, например в формате «день.месяц.год». Также библиотека позволяет работать с таймерами, расписаниями и логами.

Пример использования:

from datetime import datetime, timedelta

now = datetime.now() print(now) # текущее время
print(now.strftime("%d.%m.%Y")) # формат даты
tomorrow = now + timedelta(days=1)
print(tomorrow)
